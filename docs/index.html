Overview

One binary, the whole audit.

Point kagu scan at any git repo and it walks the entire log, checks every commit against the Conventional Commits spec, scores it out of 100, and calls out exactly which messages break the rules — and why.

~/my-repo 
$ kagu scan

Features

Everything a commit auditor should do.

No plugins. No config. No runtime. Every feature in a single 1.1 MB binary.

Full history audit

Walks the entire git log, scores it 0–100, and flags every commit that breaks the spec — with line-level reasons.

Branch-only CI scan

--since origin/main only audits new commits on your branch. Zero noise on old history.

One-command git hook

kagu hook install drops a commit-msg hook in place. Won't clobber an existing one.

Per-author breakdown

--authors surfaces who needs a refresher. Useful for onboarding audits and team retros.

Machine-readable JSON

--json pipes a fully typed report into dashboards, scripts, or CI artifact reporters.

CI-friendly exit codes

Exits 0 on clean, 1 on errors, 2 on warnings (with --strict). Drop it into any pipeline.

Quick Start

Install, scan, done.

Pick your install method and copy the command. No post-install, no config, no husky.

cargo — recommended 
# Requires Rust — install at rustup.rs
$ cargo install kagu

# Audit every commit in the current repo
$ cd ~/my-repo
$ kagu scan
cargo from git — latest, no crates.io 
$ cargo install --git https://github.com/iamkorun/kagu
from source 
$ git clone https://github.com/iamkorun/kagu.git
$ cd kagu
$ cargo install --path .
pre-built binary 
# Linux x86_64
$ curl -LO https://github.com/iamkorun/kagu/releases/latest/download/kagu-x86_64-linux.tar.gz
$ tar -xzf kagu-x86_64-linux.tar.gz
$ sudo mv kagu /usr/local/bin/
Usage

Real commands, real output.

Branch-only CI
Fail CI only on new violations
$ kagu scan --since main --quiet
# exit 0 — all clean

$ kagu scan --since origin/main
kagu scan

  ✗ 3fa9d12  fix things
      error [format] subject does not match ...

  score: 66/100
Per-author report
Surface who needs a refresher
$ kagu scan --authors

authors
  author                    total  clean  errors  score
  ada.lovelace@ex.io          14     14       0  100/100
  grace.hopper@ex.io           8      6       2   75/100
  linus.t@ex.io                2      1       1   50/100
Commit-msg hook
Catch violations at commit time
$ kagu hook install
kagu: installed commit-msg hook at .git/hooks/commit-msg

# Now try committing a bad message:
$ git commit -m "fix things"
kagu: commit message rejected
  error [format] subject does not match `<type>...`
