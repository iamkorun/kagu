What kagu calls out

Run kagu scan on any git repo. Every violation, scored and surfaced instantly.

Full history audit

Scans your entire git log, scores it 0–100, and tells you exactly which commits broke the spec — and why.

$ kagu scan
kagu scan
 
✗ a1b2c3d WIP stuff happening
error [format] subject does not match `<type>(<scope>)?!?: <description>`
! d4e5f67 feat: add database connection.
warn [punctuation] description should not end with `.`
✗ 7890abc release the thing
error [format] subject does not match `<type>(<scope>)?!?: <description>`
 
summary
total: 24 clean: 21 warnings: 1 errors: 2 skipped: 1
score: 87/100
 
types
chore 4
feat 9
fix 5
refactor 3

Branch-only CI scan

Only audits new commits on your branch. Zero noise on old history. Fails CI with exit code 1 if anything is wrong.

$ kagu scan --since main --quiet
# exit 0 — all clean
 
$ kagu scan --since origin/main
kagu scan
 
✗ 3fa9d12 fix things
error [format] subject does not match ...
 
score: 66/100

Per-author breakdown

Find out who needs a refresher. Useful for team retrospectives or onboarding audits.

$ kagu scan --authors
authors
author total clean errors score
alice@example.com 14 14 0 100/100
bob@example.com 8 6 2 75/100
carol@example.com 2 1 1 50/100

One-command git hook

Catches violations at commit time, not at PR review time. Won't clobber an existing hook.

$ kagu hook install
kagu: installed commit-msg hook at .git/hooks/commit-msg
 
# Now try committing a bad message:
$ git commit -m "fix things"
kagu: commit message rejected
error [format] subject does not match `<type>...`

Machine-readable JSON

Pipe into dashboards, scripts, or CI artifact reporters. Every violation structured and typed.

$ kagu scan --json | jq .summary
{
"total": 24,
"clean": 21,
"warnings": 1,
"errors": 2,
"skipped": 1,
"score": 87
}

Install kagu

# Requires Rust — install at rustup.rs
cargo install kagu
 
# Then run:
kagu scan
git clone https://github.com/iamkorun/kagu.git
cd kagu
cargo install --path .
# Download from GitHub Releases
# https://github.com/iamkorun/kagu/releases
 
# Linux example:
curl -LO https://github.com/iamkorun/kagu/releases/latest/download/kagu-x86_64-linux.tar.gz
tar -xzf kagu-x86_64-linux.tar.gz
sudo mv kagu /usr/local/bin/
# Install directly from the git repo (no crates.io needed)
cargo install --git https://github.com/iamkorun/kagu
