kagu

Screeches at bad commits.

A single Rust binary that audits every commit in your git log against the Conventional Commits spec. No Node.js. No config file. No husky dance.

why kagu
01
Full history audit

Walks the entire git log, scores 0–100, flags every non-conforming commit with line-level reasons.

02
Branch-only CI

--since main audits only new commits. Zero noise from old history.

03
One-command hook

kagu hook install drops a commit-msg hook. Won't clobber an existing one.

04
Per-author breakdown

--authors surfaces who needs a refresher. Useful for onboarding audits.

05
Machine-readable JSON

--json pipes a typed report into dashboards, scripts, or CI artifact reporters.

06
CI-friendly exit codes

Exits 0 on clean, 1 on errors, 2 on warnings with --strict. Drop it into any pipeline.

examples
branch-only ci
Fail CI only on new violations
$ kagu scan --since main --quiet
# exit 0 — all clean

$ kagu scan --since origin/main
kagu scan

  ✗ 3fa9d12  fix things
      error [format] subject does not match ...

  score: 66/100
per-author report
Surface who needs a refresher

commit-msg hook
Catch violations at commit time
$ kagu hook install
kagu: installed commit-msg hook at .git/hooks/commit-msg

# Now try committing a bad message:
$ git commit -m "fix things"
kagu: commit message rejected
  error [format] subject does not match `<type>...`

install
cargo recommended 
# Requires Rust — install at rustup.rs
$ cargo install kagu

# Audit the current repo
$ cd ~/my-repo
$ kagu scan
from git latest, no crates.io 
$ cargo install --git https://github.com/iamkorun/kagu
from source 
$ git clone https://github.com/iamkorun/kagu.git
$ cd kagu
$ cargo install --path .
pre-built binary 
# Linux x86_64
$ curl -LO https://github.com/iamkorun/kagu/releases/latest/download/kagu-x86_64-linux.tar.gz
$ tar -xzf kagu-x86_64-linux.tar.gz
$ sudo mv kagu /usr/local/bin/
